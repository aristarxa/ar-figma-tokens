# Token Manager - Figma Plugin

## Описание
Плагин для управления токенами (variables) в Figma с функциями:
- Просмотр токенов в виде иерархического дерева
- Перенос токенов между коллекциями
- Поиск токенов

## Установка
```bash
npm install
npm run build
```

## Разработка
```bash
npm run watch
```

## Структура проекта
```
src/
├── code.ts              # Точка входа плагина
├── ui.html              # Пользовательский интерфейс
├── types/
│   └── messages.ts      # Типы сообщений между UI и плагином
├── utils/
│   ├── treeBuilder.ts   # Построение иерархии токенов
│   └── variableHelpers.ts # Вспомогательные функции для работы с переменными
└── services/
    └── variablesService.ts # Бизнес-логика работы с Figma Variables API
```

## Использование
1. Откройте файл в Figma
2. Запустите плагин: Plugins → Development → Token Manager
3. Выберите коллекцию токенов
4. Используйте поиск или выбирайте токены для переноса
